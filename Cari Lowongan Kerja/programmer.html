Lowongan Kerja Programmer - Onklass ID

Tanggung Jawab Pekerjaan :

  • Mengembangkan dan memelihara aplikasi berbasis web dan mobile
  • Menulis kode yang efisien, terstruktur, dan dapat di-maintain
  • Berkolaborasi dengan tim desain dan pengembang lainnya
  • Menganalisis dan memperbaiki bug atau masalah pada aplikasi
  • Mengoptimalkan kinerja aplikasi untuk pengalaman pengguna yang lebih baik

Keahlian :

  • Memahami bahasa pemrograman seperti PHP, JavaScript, atau Python
  • Berpengalaman dengan framework seperti Laravel, React, atau Vue.js
  • Menguasai database MySQL/PostgreSQL
  • Familiar dengan Git dan sistem version control
  • Memahami konsep API dan integrasi dengan third-party services

Kualifikasi :

  • Pendidikan minimal D3/S1 di bidang Teknik Informatika, Sistem Informasi, atau terkait
  • Pengalaman minimal 2 tahun di bidang pemrograman (Fresh Graduate dipersilakan melamar)
  • Mampu bekerja dalam tim maupun secara individu
  • Memiliki pemahaman logika pemrograman yang baik
  • Diutamakan yang berdomisili di Jakarta atau sekitarnya

Waktu Bekerja :

Senin - Jumat, Jam 08.00 - 17.00 | Sabtu, Jam 08.00 - 15.00

Tunjangan :

  • Bonus berdasarkan kinerja
  • Asuransi kesehatan
  • Makan siang disediakan

Insentif :

  • Liburan ke luar negeri bagi karyawan berprestasi
  • Pelatihan dan sertifikasi profesional
